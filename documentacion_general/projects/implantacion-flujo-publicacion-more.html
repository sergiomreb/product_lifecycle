Implantación flujo publicación en cadenas

Ir a JIRA → ← Volver

Resumen

Esta iniciativa tiene como objetivo migrar el concepto de publicación de los componentes comerciales, dejando de usar las exclusiones y el buyable a ese nivel en favor de usar el milestone de publicación asociado.

El cambio permitirá una mejor sincronización entre los sistemas de publicación y una gestión más centralizada del ciclo de vida de los productos en cadenas.

Actualmente se encuentra en fase de implantación, con avances significativos en la integración de los nuevos flujos.

Decisiones pendientes

  • Definir la estrategia de migración progresiva de datos históricos
  • Validar impacto en sistemas downstream dependientes de las exclusiones actuales
  • Establecer SLAs para la publicación en milestone de publicación
  • Planificar ventanas de mantenimiento para migración de datos
  • Documentar cambios en API de publicación para clientes internos

Temas pendientes completar

  • Finalizar documentación de cambios en API de publicación
  • Completar validación de impacto en sistemas downstream
  • Cerrar issues pendientes en JIRA
